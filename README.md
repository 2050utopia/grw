> 个人事务管理系统是一套可以管理你所有的事务的系统程序，他操作方便、功能强大并且你可以免费使用它。http://www.grw.name

#### 微信功能

> 新版本(V0.2)将增加微信功能，你可以先关注我们微信账号获取最新进展！

```
1.通过微信随时记录每天发生的事情(文字、图片、语音甚至图像)
2.通过微信处理人际关系信息
3.通过微信记录每天的行走的路径
4.通过微信处理待办事项
5.通过微信查看密码保险箱密码
6.更多功能敬请期待!
```
